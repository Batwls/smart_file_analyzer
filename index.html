📊 Smart File Analyzer

A powerful command-line tool for analyzing text files.

📌 About

Smart File Analyzer is a simple yet powerful CLI tool that helps you analyze text files. It provides insights such as word counts, unique word analysis, longest words, and most common words.

🚀 Usage

Run the following command to analyze a file:

smart_file_analyzer example.txt

Example output:

📊 Smart File Analyzer Results for: example.txt
---------------------------------
📄 Total Lines: 4
📝 Total Words: 16
🔠 Total Characters: 83
📏 Longest Word: programming
🏆 Most Common Word: "Rust" (4 times)
🔣 Unique Words: 12
